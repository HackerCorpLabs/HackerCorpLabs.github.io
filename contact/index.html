FAQ

How can I contribute to a project?

Check out our GitHub repositories and read the CONTRIBUTING.md file. We welcome pull requests, bug reports, and feature suggestions.

Do you offer support for your projects?

Yes! Join our Discord server for community support or open an issue on GitHub for technical problems.

Can I use your projects commercially?

Most of our projects are open source under permissive licenses. Check the specific project's LICENSE file for details.
